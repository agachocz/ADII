# ADII
This repository is created for the purpose of the reconstruction of the ASEAN Digital Integration Index (ADII) for a wider geographical scope. ADII index covers a broad definition of digital development, including institutional and infrastructural readiness, environment of innovation, human skills, data protection and cybersecurity, and digital trade, payments and logistics. Its benefit lies in the use of open and international data sources (such as World Bank, UNESCO, ITU Data Hub, among others), which cover many countries outside of the region, offering high quality and reliable data.
